# IVC心脏参数测量说明

## 概述

本数据集包含超声心动图（IVC视图，下腔静脉切面）中1个关键参数测量。参数由两个点（p1, p2）的坐标定义，通过计算两点之间的距离得到测量值。

---

## CSV文件格式

| 列名 | 说明 |
|------|------|
| `image_path` | 图像文件路径 |
| `IVC_p1_xy` | 下腔静脉 - 点1坐标 [x, y] |
| `IVC_p2_xy` | 下腔静脉 - 点2坐标 [x, y] |
| `height` | 图像高度（像素） |
| `width` | 图像宽度（像素） |
| `task_name` | 任务名称（Regression） |
| `num_classes` | 类别数（2） |
| `task_id` | 任务标识（IVC） |

---

## 1个测量参数详解

### IVC (下腔静脉)
- **对应列**: `IVC_p1_xy`, `IVC_p2_xy`
- **测量方法**: 计算点1与点2之间的欧氏距离
- **临床意义**: 下腔静脉直径，用于评估血容量状态、右心房压力、脱水状态等。下腔静脉直径随呼吸变化，吸气时变窄，呼气时增宽，可用于指导液体管理
