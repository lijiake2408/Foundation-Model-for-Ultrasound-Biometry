# PSAX心脏参数测量说明

## 概述

本数据集包含超声心动图（PSAX视图，胸骨旁短轴切面）中2个关键参数测量。每个参数由两个点（p1, p2）的坐标定义，通过计算两点之间的距离得到测量值。

---

## CSV文件格式

| 列名 | 说明 |
|------|------|
| `image_path` | 图像文件路径 |
| `RVOT_p1_xy` | 右心室流出道 - 点1坐标 [x, y] |
| `RVOT_p2_xy` | 右心室流出道 - 点2坐标 [x, y] |
| `PA_p1_xy` | 肺动脉 - 点1坐标 [x, y] |
| `PA_p2_xy` | 肺动脉 - 点2坐标 [x, y] |
| `height` | 图像高度（像素） |
| `width` | 图像宽度（像素） |
| `task_name` | 任务名称（Regression） |
| `num_classes` | 类别数（4） |
| `task_id` | 任务标识（PSAX） |

---

## 2个测量参数详解

### 1. RVOT (右心室流出道)
- **对应列**: `RVOT_p1_xy`, `RVOT_p2_xy`
- **测量方法**: 计算点1与点2之间的欧氏距离
- **临床意义**: 右心室流出道的直径，用于评估右心室流出道梗阻、肺动脉狭窄等疾病

### 2. PA (肺动脉)
- **对应列**: `PA_p1_xy`, `PA_p2_xy`
- **测量方法**: 计算点1与点2之间的欧氏距离
- **临床意义**: 肺动脉主干直径，用于评估肺动脉高压、肺动脉扩张等疾病
